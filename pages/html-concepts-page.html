Definición y características de HTML 5

En este sitio web encontrarás información sobre la definición y características de HTML 5. Asi como también, las etiquetas más importantes que deberias usar en tu web.

¿Qué es HTML?

HTML, o HyperText Markup Language (Lenguaje de Marcado de Hipertexto), es el estándar utilizado para crear y estructurar páginas web. Es un lenguaje de marcado que define la estructura básica de una página web, utilizando etiquetas para organizar y presentar contenido como texto, imágenes, videos, enlaces, entre otros elementos.

* Este video es una elaboración propia creada con el propósito de explicar la importancia de HTML y sus etiquetas en la creación de páginas web.

¿Qué significa que sea "5"?

La especificación HTML5 es la quinta revisión importante del lenguaje básico de la World Wide Web, el HTML. Aunque la "5" no se refiere a una versión específica de HTML, sino más bien a una evolución continua del estándar, se adoptó el nombre "HTML5" como una forma simple de referirse a esta versión más moderna y avanzada del lenguaje de marcado.


HTML5 introdujo una serie de nuevas características y funcionalidades destinadas a mejorar la experiencia de desarrollo web y la interoperabilidad entre navegadores. Algunas de las adiciones más significativas incluyen elementos semánticos para una mejor estructuración del contenido, capacidades de reproducción de audio y video sin necesidad de complementos externos, capacidades de gráficos 2D y 3D junto con WebGL, entre muchas otras.
